# SPHY CSV Quantum Renderer

Interactive 3D visualizer for SPHY quantum datasets using py5.

The renderer transforms quantum CSV logs into animated spatial structures, allowing researchers to visually explore:

* state evolution;
* coherence;
* energetic distribution;
* temporal trajectories;
* vector dynamics;
* emergent behavior in massive datasets.

---

# Overview

The visualizer:

* opens any CSV containing `Q*` columns;
* automatically detects quantum state fields;
* converts data into a 3D point cloud;
* renders trajectories and vectors in real time;
* creates an interactive spatial exploration environment.

The system was designed to work with datasets generated by frameworks such as:

* Qiskit
* Cirq
* PennyLane
* Amazon Braket
* Strawberry Fields

---

# Main Features

## 📂 Graphical CSV Loader

At startup, a graphical file picker allows users to select a dataset:

```text
Select SPHY CSV
```

No manual path editing required.

---

# 🌌 Real-Time 3D Rendering

Each CSV row is converted into:

* spatial position;
* directional vector;
* energy intensity;
* temporal frame.

Particles are rendered in real-time using OpenGL acceleration through py5.

---

# ⚛️ Visual Interpretation

Columns:

```text
Q1 Q2 Q3 ...
```

are interpreted as spatial coordinates of the quantum state.

Mapping:

| Column | Axis |
| ------ | ---- |
| Q1     | X    |
| Q2     | Y    |
| Q3     | Z    |

---

# 🌊 Energy Mapping

Visual intensity is derived from the absolute mean of all Q values:

```python
energy = np.mean(np.abs(q_values))
```

Energy controls:

* brightness;
* sphere size;
* color intensity.

---

# 🧭 Dynamic Vectors

Each frame receives a procedural vector:

```python
vx = np.sin(i * 0.02)
vy = np.cos(i * 0.02)
vz = np.sin(i * 0.01)
```

These vectors represent temporal/dynamic orientation.

---

# 🌀 Temporal Trails

The renderer can draw historical trajectories of previous frames, allowing visualization of:

* recurrence;
* convergence;
* dispersion;
* geometric stability.

---

# 🎥 Visualization Modes

## Playback Mode

Renders one frame at a time.

Ideal for temporal analysis.

---

## Show All Mode

Renders all particles simultaneously.

Ideal for observing large-scale spatial structures.
